This package uses pnpm, NOT npm, as it's package manager.
It also uses jj instead of git as its version control system - see your jujutsu skill if you need to run jj/git commands.
Pull Requests can be managed with `jj spr` - see the jj-spr skill
The PRD is Github issue 1, you can use the gh cli to interact with it.
other issues can be found with `gh issue list`.
