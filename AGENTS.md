This project is a wrapper around convex workflow that intergrates it with confect, a libary that allows convex functions to be writtin with Effect.ts
The repo is a monorepo, with the npm package at `packages/confect-workflow` and an example site using the package at `apps/example`.
We are using Vite plus (on top of pnpm) instead of npm. Read docs/VITE_PLUS.md for more details
The specification for how the confect-workflow package is meant to work is at docs/CONFECT_WORKFLOW_SPEC.md
